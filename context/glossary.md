# Glossary

Project-specific terms, abbreviations, and conventions.

## Terms

### Course
A structured learning experience consisting of lessons, assessments, and learning objectives. The deliverable unit that learners engage with.

**Not to be confused with:** Curriculum (the broader design and structure)
**See also:** Curriculum, Lesson

---

### Curriculum
The overall design, structure, and strategy for what should be taught. Includes learning objectives, competency progressions, and pedagogical approach.

**Not to be confused with:** Course (the specific implementation of a curriculum)
**See also:** Course, Competency

---

### Agent
An LLM-powered assistant with a specific role in the teaching/learning process. Different agent types serve different purposes (teaching, coaching, assessment).

**Types identified so far:**
- Teaching agent: Delivers content, explains concepts
- Coaching agent: Guides learner practice, provides feedback
- Assessment agent: Evaluates learner understanding

---

### Portable Format
The export format that allows courses to run in a separate web application. Details TBD.

**See also:** Export

## Domain-Specific Vocabulary

### Education/Learning

| Term | Meaning |
|------|---------|
| Competency | A measurable skill or knowledge area |
| Progression | The path from beginner to mastery in a competency |
| Scaffold | Temporary support that helps learners reach understanding |
| Formative assessment | Ongoing assessment during learning |
| Summative assessment | Final evaluation of learning |
